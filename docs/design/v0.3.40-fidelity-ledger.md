## Above-the-fold copy differences

- Reference next action: `Complete Windows printer setup in Step 1 below.`
- Implemented next action while the listener is already healthy: `Connect your POS in Step 2 or send a Test Receipt in Step 3.`
- Reason: the production page uses live listener health, so it does not tell a customer to repeat setup after a listener is already running.
- Reference sample connection address: `127.0.0.1`
- Implemented connection address in the saved render: `EPEREZP11`
- Reason: Simple Mode displays the actual listener connection target supplied by the service. Loopback listeners are normalized to `127.0.0.1`; network-accessible listeners keep their real host/address.

## Interaction verification

- Simple Mode is the first-use default.
- Changing to Expert Mode survives a page reload through per-user local storage.
- Returning to Simple Mode retains the selected/latest receipt.
- **Send test receipt** creates and renders the built-in test job immediately.
- **Open setup wizard** opens Settings at Printer Setup Wizard.
- **Run diagnostics** opens Settings at Support and exposes Connection Diagnostics.
- No browser console warning or error was recorded during the tested flow.
